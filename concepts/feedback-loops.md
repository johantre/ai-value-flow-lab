# Feedback Loops

Mechanisms where output becomes input, creating circular causality. Reinforcing loops amplify change (virtuous or vicious cycles). Balancing loops resist change (stabilizers). AI dramatically accelerates feedback loop speed — a key reason AI-native orgs work differently.
