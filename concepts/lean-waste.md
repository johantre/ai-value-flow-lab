# Lean Waste (TIMWOOD + Talent)

The 8 wastes identified in Lean: **T**ransport, **I**nventory, **M**otion, **W**aiting, **O**verproduction, **O**verprocessing, **D**efects, Unused **Talent**. Any activity that consumes resources without creating customer value. Core lens for value stream analysis.
