# Complexity (Cynefin)

In the Complex domain, cause and effect can only be understood in retrospect. The appropriate response is [[probe-sense-respond|Probe → Sense → Respond]], not plan-and-execute. Most organizational transformations are Complex. See [[cynefin-domains]].
