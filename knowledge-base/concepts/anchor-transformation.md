# Anchor Transformation

The 1–3 highest-leverage redesign points in an organization where AI-native redesign changes the fundamental economics or possibilities. Anchor transformations pull other changes along. Criteria: changes unit economics significantly, generates data for further AI, removes a bottleneck, or builds credibility. From [[toward-ai-native-ai-first]].
