# Value Stream Mapping

A Lean technique for visualizing the end-to-end flow of value creation from customer need to delivery. Maps steps, inputs, outputs, roles, wait times, and types of work. Always done **[[outside-in]]** — starting from the customer. Foundation of the Value Stream Assessment Template.
