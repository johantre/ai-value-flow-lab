# Learning Organization

Senge's concept: an organization that continuously learns and adapts. Built on five disciplines: [[systems-thinking|systems thinking]], personal [[mastery]], [[mental-models|mental models]], shared vision, team learning. The sensing cadence IS a learning loop. AI-first transformation requires organizational learning capability.
