# SCARF Model

David Rock's five social threat/reward domains: **S**tatus, **C**ertainty, **A**utonomy, **R**elatedness, **F**airness. Threats to any trigger the same brain response as physical danger. AI-first transitions threaten all five — design the transition to minimize SCARF threats.
